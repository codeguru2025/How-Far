## Privacy Policy Summary (for Data Safety)

**Data Collected:**
- Phone number (required for registration)
- Name (for user identification)
- Location (for ride matching and tracking)
- Payment information (processed by PayNow)
- Transaction history (for wallet functionality)

**Data Sharing:**
- With other users: Name only (for identification)
- With payment processor: Payment details only
- We do not sell user data to third parties

**Data Security:**
- All data encrypted in transit (TLS)
- Passwords cryptographically hashed
- Data stored on secure cloud servers (Supabase)

**User Rights:**
- Users can request data deletion via app settings
- Users can download their data upon request
- Contact: privacy@howfar.app
